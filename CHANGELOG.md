# Changelog

## v2.0.0 (2026-06-30)

### 重大重构：从概率辩论到因子分析

核心方法论从"六专家各自给出全场概率估计 + 圆桌辩论"升级为"六专家各负责一个因子层 + 因子聚合公式"。

### 第四步：六专家因子分析（替代原概率估计）

每位专家只负责自己分配的因子层，输出标准化因子调整值（±Xpp per sub-factor），不再各自输出全场概率：

| 专家 | 因子层 | 子因子数 |
|------|--------|---------|
| 西尔弗 | 基础实力 | 5（Elo差/进攻效率/防守效率/转化率/定位球） |
| 温格 | 人员+战术 | 10（人员5+战术5） |
| 莫布森 | 激励/情境 | 6（出线压力/轮换动机/首轮冷门反弹/新军效应/非洲韧性/已出线强队末轮） |
| 贝纳姆 | 市场结构 | 3（跨盘矛盾/vig异常/散户偏差） |
| 塔勒布 | 尾部风险 | 2（红牌风险/早期进球崩塌）+ 尾部预留% |
| 泰特洛克 | 校准 | CI展宽/收窄系数+信心度 |

### 第五步：因子交叉验证（替代圆桌辩论）

- 同向检查：5位因子专家方向一致性判断
- 对冲检测：贝纳姆市场层与其他四层方向相反时→定价矛盾→额外关注
- 塔勒布尾部预留>15% + vig>8% → 自动降仓 50%
- 泰特洛克校准系数应用到合成概率 CI

### 第六步：因子聚合（替代概率合成）

固定公式取代主观综合：
```
基准概率 = 市场隐含概率（去 vig）
最终概率 = 基准 × (1 + Σ(各层净调整 × 层权重)) × (1 - 尾部预留%) × CI系数
层权重：基础实力30% / 人员+战术25% / 激励15% / 市场15% / 尾部扣减
```

### 第十一步：因子级赛后复盘（替代原赛后追踪）

- 新增因子贡献回溯：逐场追溯每个子因子预测方向 vs 实际，识别"错在哪层"
- 反事实 Brier：计算"如果去掉某层因子后的 Brier"——识别拖累层
- 因子增删改（每轮必做）：连续3轮错误率>60%→建议删除；新 pattern→建议新增
- 权重动态调整：基于近3轮命中率变化调整层权重

## v1.1.0 (2026-06-16)

### 第十一步扩展

从 5 行清单扩展为 8 个子步骤的完整赛后复盘系统：
- 11.1 接收赛果 / 11.2 对照预测 / 11.3 审计表决对照
- 11.4 专家表现评估 / 11.5 提取经验教训 / 11.6 Brier 滚动监控
- 11.7 skill 自迭代 / 11.8 更新追踪文件

### 新增机制

- Brier 阈值：<0.15 绿 / 0.15-0.25 黄 / >0.25 红
- 红区自动修正：合成概率下调 5pp，80% CI 展宽 1.5×
- 连续两轮红区→暂停下注
- 每三轮自动审查概率校准、专家权重、过滤器参数
